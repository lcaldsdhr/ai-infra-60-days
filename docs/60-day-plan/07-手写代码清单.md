### Q5：手写 AdamW
- **难度**：简单
- **建议行数**：30 行
- **题目**：30 行 Python 实现 AdamW
- **要点**：
  - 一阶矩 `m`、二阶矩 `v`
  - bias correction
  - **decoupled weight decay**（区别于 L2 正则）
- **参考骨架**：
  ```python
  class AdamW:
      def __init__(self, params, lr=1e-3, betas=(0.9, 0.999), eps=1e-8, weight_decay=0.01):
          ...
      def step(self):
          for p in self.params:
              m, v = self.state[p]['m'], self.state[p]['v']
              m = betas[0] * m + (1 - betas[0]) * p.grad
              v = betas[1] * v + (1 - betas[1]) * p.grad ** 2
              m_hat = m / (1 - betas[0] ** t)
              v_hat = v / (1 - betas[1] ** t)
              p.data -= lr * (m_hat / (v_hat.sqrt() + eps) + weight_decay * p.data)
  ```
- **对应知识**：优化器

### Q17：手写 LoRA 层
- **难度**：简单
- **建议行数**：30 行
- **题目**：手写 LoRA（含 `merge_and_unload`）
- **要点**：
  ```python
  class LoRALinear(nn.Module):
      def __init__(self, base, r=8, alpha=16):
          super().__init__()
          self.base = base
          self.lora_A = nn.Parameter(torch.randn(r, base.in_features))
          self.lora_B = nn.Parameter(torch.zeros(base.out_features, r))
          self.scale = alpha / r
      def forward(self, x):
          return self.base(x) + (x @ self.lora_A.T @ self.lora_B.T) * self.scale
  ```
  + 实现 `merge_and_unload`：把 LoRA 融合回 base.weight
- **对应知识**：PEFT

## 四、训练流程模板（手写一遍胜过读 10 遍）

> 把以下 30 行训练 step 当作"必背模板"，**理解每一行的存在原因**。

```python
# 一个完整的 FSDP / 手动训练 step 模板
for epoch in range(num_epochs):
    for batch in dataloader:
        # 1) 前向
        with torch.autocast(device_type='cuda', dtype=torch.bfloat16):
            loss = model(**batch)
        
        # 2) 反向
        loss.backward()
        
        # 3) 梯度裁剪
        if max_grad_norm > 0:
            torch.nn.utils.clip_grad_norm_(model.parameters(), max_grad_norm)
        
        # 4) 优化器步进
        optimizer.step()
        
        # 5) 梯度清零
        optimizer.zero_grad(set_to_none=True)
        
        # 6) LR 调度
        lr_scheduler.step()
```

> 面试常考变体：
> - "如果用 FSDP，每一步在做什么？"
> - "如果用 ZeRO-3，第 2 步和第 4 步分别会发生什么通信？"
> - "如果用 PP，第 1 步是跨 rank 的吗？"
> - "如果用 gradient accumulation，循环要包几层？"
